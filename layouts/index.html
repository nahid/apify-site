Key Features

  • Comprehensive API Testing

    Define and run detailed API tests with support for multiple request methods (GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS), rich payload types (JSON, Text, Form Data), file uploads, and detailed assertions for response validation.

  • Integrated Mock Server

    Simulate API endpoints with dynamic and conditional responses. Use template variables to generate realistic mock data, and manage mock definitions in simple file-based configurations.

  • ES6(JavaScript) Support for Dynamic Scripting

    Leverage ES6(JavaScript) features like arrow functions, template literals, and destructuring to create more expressive and maintainable test scripts.

  • Environment Management

    Manage different configurations for development, staging, and production environments with project, environment, and request-level variable overriding.

  • User-Friendly CLI

    Interactively create test and mock definitions, view detailed reports with visual progress indicators, and deploy with a single executable file.

  • Deployment & Extensibility

    Enjoy a simplified deployment with a single executable file, making it easy to get started and integrate into your testing workflows.
