Apify CLI

A robust command-line tool for API testing and validation.

Get Started GitHub

Apify is a robust command-line tool designed for API testing and validation. It allows developers to define API tests in JSON format and execute them against endpoints, providing detailed output of the request, response, and test results. The tool offers centralized environment management, variable substitution, and support for various payload types and file uploads. It also includes a powerful mock server for simulating API responses during development and testing.

API Testing

Define, execute, and validate your API endpoints with comprehensive test definitions and assertions.

Mock Server

Simulate API responses for efficient development, testing, and rapid prototyping without backend dependencies.

Key Features

  • Comprehensive API Testing

    Define and execute API tests with detailed request, response, and test results.

  • Centralized Environment Management

    Easily manage different environments (development, staging, production) for your API tests.

  • Flexible Variable Substitution

    Use variables to make your tests dynamic and reusable across different environments and scenarios.

  • Support for Diverse Payload Types

    Test APIs with various data formats, including JSON, plain text, and form data.

  • File Upload Capabilities

    Seamlessly test endpoints that require file uploads.

  • Powerful Mock Server

    Simulate API responses for efficient development and testing, even when the actual API is unavailable.

  • Streamlined Workflow

    Automate your API testing process, saving time and reducing manual effort.
